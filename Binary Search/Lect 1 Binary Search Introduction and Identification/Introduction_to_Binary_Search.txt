Keywords to identify Problems based on the Binary Search.

1. Problem is related to Sorted Array.
2. Find a Required Value


Keywords to identify Problems based on the Binary Search on Answer.

1. Problem is mostly related to the Unsorted Array.
2. Find a Required Minimum (Floor) or Maximum (Ceil) Value that Satisfies Condition


Binary Search Problem Types:

  Sr |   Array or String  |           Type           |
  1. |       Sorted       |      Binary Search       |
  2. |  Partially Sorted  |  Modified Binary Search  |
  3. |      UnSorted      |  Binary Search on Answer |
